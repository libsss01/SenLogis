# SenLogis

SenLogis is a web project focused on housing/rental management and real estate presentation.

The project is currently evolving and serves as a practical way to improve web development, project organization, and application structure.

## Project Goal

The goal of SenLogis is to explore how a digital platform can help organize housing or rental-related information through a simple web interface.

Potential use cases:

- Present housing offers
- Organize rental/property information
- Improve visibility for real estate services
- Practice real-world web application structure

## Tech Stack

The repository currently contains web assets and templates. The technical stack may evolve as the project becomes more structured.

Planned or possible technologies:

- HTML
- CSS
- JavaScript
- PHP or another backend technology
- MySQL or another database system
